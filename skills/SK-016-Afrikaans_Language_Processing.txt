2. REGIONAL DIALECT RECOGNITION
-------------------------------

CAPE AFRIKAANS (Kaapse Afrikaans):
Characteristics:
- Influenced by Malay, Portuguese, Khoi languages
- Distinctive vocabulary: "kamma" (as if), "bra" (brother)
- Pronunciation variations
- Often more melodic intonation

Common markers:
- "djy" instead of "jy" (you)
- "aweh" - greeting/acknowledgment
- "lekka" instead of "lekker"

TRANSVAAL AFRIKAANS:
Characteristics:
- Considered "standard" Afrikaans
- Basis for formal written Afrikaans
- Used in media and education

EASTERN CAPE / BORDER:
Characteristics:
- English influence more prominent
- Distinctive pronunciation patterns

FREE STATE AFRIKAANS:
Characteristics:
- More conservative forms
- Agricultural terminology
- Rural expressions

NORTHERN CAPE / NAMAQUALAND:
Characteristics:
- Nama/Khoi influence
- Unique vocabulary items
- Distinctive phonology

3. COMMON SPEECH PATTERNS & FILLERS
-----------------------------------

DISCOURSE MARKERS:
- "nÃ©" / "ne" - tag question (isn't it?, right?)
- "ja-nee" - literally "yes-no", means "well" or "indeed"
- "mos" - emphasis particle (you know, obviously)
- "sommer" - just, simply, for no particular reason
- "tog" - after all, though, surely
- "dan" - then (often as filler)
- "nou" - now (often as filler: "nou ja")
- "maar" - but/just (often softening)
- "eintlik" - actually
- "basies" - basically (from English)

INTERJECTIONS:
- "Ag" / "Ach" - oh well, expression of resignation
- "Ai" / "Eina" - ouch, expression of pain/sympathy
- "Sjoe" / "Tjoe" - wow, expression of surprise
- "Haai" - hey, expression of dismay
- "Jislaaik" - expression of surprise
- "Yoh" - expression of amazement

AFFIRMATIONS:
- "Ja" - yes
- "Ja-nee" - yes (emphatic)
- "Reg" / "Reggies" - right, correct
- "Okay" / "Orraait" - okay (anglicism)

4. CODE-SWITCHING PATTERNS
--------------------------

TYPE 1 - INSERTIONAL:
Single word/phrase insertion from other language
"Die meeting is more important as ons gedink het."
"We need to finaliseer this by Friday."

TYPE 2 - ALTERNATIONAL:
Full clause switching
"Ek dink we should postpone. Dit maak meer sin."
"Let me explain. Kom ek sÃª dit in Afrikaans."

TYPE 3 - CONGRUENT LEXICALISATION:
Shared grammatical structure
"Ek het die document ge-email." (email â†’ ge-email)
"Hulle moet dit update." (update used as Afrikaans verb)

COMMON HYBRID FORMS:
- ge- + English verb: ge-email, ge-update, ge-post, ge-check
- English noun + Afrikaans plural: files â†’ "files" or "faile"
- English adjective + Afrikaans: "Dit is so nice"

TRANSCRIPTION RULES:
1. Preserve exactly as spoken
2. Use "ge-" prefix as spoken
3. Do not "correct" to pure Afrikaans
4. Do not translate

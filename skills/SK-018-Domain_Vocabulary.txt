================================================================================
VOCABULARY EXPANSION PROTOCOL
================================================================================

PRE-TRANSCRIPTION:
1. Request client/project context from user
2. Check for existing vocabulary list for client
3. Add any mentioned proper nouns to session vocabulary
4. Note unfamiliar terms for post-processing verification

DURING TRANSCRIPTION:
1. Flag potential proper nouns (capitalised in context)
2. Note repeated technical terms
3. Mark uncertain spellings with [?]
4. Cross-reference against vocabulary lists

POST-TRANSCRIPTION:
1. Search-replace known phonetic confusions
2. Verify proper noun consistency
3. Update vocabulary list with new confirmed terms
4. Document any unresolved terms

VOCABULARY FILE MAINTENANCE:
- Review after each major transcription project
- Add new client vocabularies as needed
- Update personnel lists as team changes
- Mark deprecated terms (e.g., former employees)
